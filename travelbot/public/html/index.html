designed by GROUPCHAT and released full machine's potential.

Beautiful Brief Fast Personal

While providing a very strong kernel, it has beautiful, brief, and good user experience features and make full use of the machine's potential with multimedia input and output forms.

Software Architecture

First of all, our system has two main components(UI for user and AI technology for chat bot which providing beneficial information to user). In order to transfer data with relational database we are using special API provided by pymysql. For the smart chat bot, adding artificial intelligence technology involved some functions provided by tensorflow library. Finally, using flask in python to transfer data with user on the website is the most efficient method.

Aim and Epics

To improve the user experience, our product will use chatbot to help the clients, it is not necessary for them to give much information about the place they want to go, they can even just give a picture for the chatbot to identify, and the chatbot will give back the detailed information about the place, additionally, the clients can finish booking the tickets by talking to the chatbot , there is no need for them to know how to operate.
